# Projeto Finanças

- PROJETO PESSOAL: EM DESENVOLVIMENTO -

Este é um projeto pessoal de controle de finanças, integrado com banco de dados MySQL, onde o usuário cadastrado, poderá fazer o cadastro de seus gastos (cadastrando preço. descrição, loja, categoria, prioriedade: alta, média e baixa).

Feito os cadastros, o intuito do sistema é apresentar em gráficos e tabelas as despesas, e ainda uma visão rápida de gastos por mês, comparação com mês anterior, etc.

Ferramentas utilizadas: C# (back-end), HTML, JavaScript, CSS, Bootstrap, MySQL.
